Notification demo steps:

1. on pandora and minion start r2dtwo if not already running

2. on minion:
 - start json receiver:
> python3 json_receiver/json_udp_receiver.py 10.10.10.2 6000

3. on pandora and minion enable pull notifications:
> telnet localhost 8000
notif_pull enable

4. on plague: 
 - generate ping traffic
> ping 10.10.11.2

Check pull notifications.

5. Failing one of the paths in mininet:
mininet> terror ip link set dev ens2f0 down

Check pull notifications.

6. Restorin the failed path in mininet:
mininet> terror ip link set dev ens2f0 up

Check pull notifications.

7. on pandora and minion disable pull notifications:
> telnet localhost 8000
notif_pull disable

8. Initiate trigger for push notifications:
In pandora telnet:
notif_trigger path2_pandora_rx c_pandora_rx 3 -n 10 -i 0.2

Check push notifications.
